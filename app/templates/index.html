Mode d'emploi de l'api


Pour le matériel RDV sur Postman et utiliser l'adresse : 127.0.0.1:5000/materiel/

  • Methode GET : pas besoin d'ajouter quoi que ce soit
  • Methode POST : {"nom":"xxx","dim":"xxx", "etat":"xxx"}
  • Methode PUT : {"nom":"xxx","dim":"xxx", "etat":"xxx", "id":"xxx"}
  • Methode DELETE :{"id":"xxx"}

Pour les employés RDV sur Postman et utiliser l'adresse : 127.0.0.1:5000/employe_informatique/

  • Methode GET : pas besoin d'ajouter quoi que ce soit
  • Methode POST :{"nom":"xxx","prenom":"xxx", "age":xxx, "profession":"xxx"}
  • Methode PUT :{"nom":"xxx","prenom":"xxx", "age":xxx, "profession":"xxx", "id":xxx}
  • Methode DELETE :{"id":"xxx"}


Il est possible de visualiser le résultat des requêtes:
  • directement sur postman
  • en saisissant les adresses correspondantes aux tables de données directement dans le navigateur
  • en rafraîchissant la page
